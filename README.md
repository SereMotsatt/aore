# Aore (API avanzada de mineral)

Esta API tiene la capacidad de reducir tu trabajo, ordenarlo y customizarlo.
Unos creadores pueden hacer decoraciones u otro tipo y otros creadores pueden poner esas creaciones dentro de sus minerales.
Está apoyado en idea de no solo crear minerales sino expandir su uso dentro del juego

## ¿Ya quieres comenzar a crear?
Lee api_en.txt si hablas ingles o api_es.txt si hablas español, pronto añadiré mas idiomas.
