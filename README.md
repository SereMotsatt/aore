# Aore (API avanzada de mineral)

Esta API tiene la capacidad de **reducir tu trabajo**, ordenarlo y customizarlo.<br/>
Unos creadores pueden hacer decoraciones u otro tipo y otros creadores pueden usar esas creaciones dentro de sus minerales gracias a que la **API** es modular.<br/>
Está apoyado en idea de no solo crear minerales sino expandir su uso dentro del juego.<br/>

## ¿Ya quieres comenzar a crear?
Lee [api_en.md](https://github.com/SereMotsatt/aore/blob/master/api_en.md) si hablas ingles o [api_es.md](https://github.com/SereMotsatt/aore/blob/master/api_es.md) si hablas español, pronto añadiré mas idiomas.<br/>
Pronto se hará la documentación completa sobre los ayudantes.
